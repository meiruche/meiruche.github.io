Research

I am interested in system and software security, mobile security, web security, program analysis, testing and model checking.

My current work includes testing and debugging of large-scale software programs (see ICSE 2015, ICST 2015, ACM TODEAS 2015) and concurrent errors detection (see FSE 2016). I am also working on secure vulnerability analysis and exploiting for Android systems (see Mobisys 2017 and ICSE2017).
